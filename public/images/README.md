## Temporary Placeholders

The code includes fallback images from Unsplash if the local images are not found. This allows the site to work immediately while you gather the proper images.

## Image Specifications

- **Logos**: PNG with transparent background, 500x500px minimum
- **Stadium Photos**: JPG, 1920x1080px minimum, landscape orientation
- **Portrait**: JPG, 800x1000px minimum
- **Trophy**: JPG, 800x800px minimum
